Структура файла на вывод:
load n  // число страничных прерываний

map {0,1}1 {0,1}2 {0,1}3 ... {0,1}m  //  список замещений страниц 
(при каждом обращении к страницам:  0 - если было страничное прерывание, 1 - если не было)
load_page x1 x2 x3 ... xm  // номера подгруженных страниц (на каждом шаге обращений 
к страницам памяти: "-1", если страница не подгружалась;  номер страницы, если таковая была подгружена)
unload_page y1 y2 y3 ... ym  // номера выгруженных страниц (то же самое, что load_page, только для выгруженных
страниц)
